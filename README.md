# 🌱 GreenCode AI Pro

## Sustainable Code Intelligence Platform

GreenCode AI Pro is a Streamlit-based sustainability dashboard that helps developers analyze code quality, monitor system resources, estimate carbon emissions, and receive optimization recommendations for greener and more efficient software development.

## 📌 Project Overview

GreenCode AI Pro is designed to promote sustainable software engineering by analyzing source code and machine learning training scripts for inefficiencies that may increase energy consumption, carbon emissions, and infrastructure costs.

The platform provides:

* Code analysis
* Sustainability scoring
* Carbon footprint estimation
* Resource monitoring
* Optimization recommendations
* User authentication
* Report generation
* GitHub repository analysis

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ShaikMuskan-0066/GreenCode-AI-Pro.git
cd GreenCode-AI-Pro
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
